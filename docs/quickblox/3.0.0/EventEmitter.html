Class: EventEmitter

EventEmitter

<private> new EventEmitter()

Creates an instance of EventEmitter.

Example
let ee = new EventEmitter();

function handleCustomEvent(data) {
 console.log(Incoming data, data);
}

ee.addListener('customEvent', handleCustomEvent);

ee.emit('customEvent', {
 'cutom': 'field'
});
// emit return this for create a chain
ee.emit('event1', {'some': 'data'}).emit('event2', {'some': 'data'});

Methods

addListener(eventName, listener)

Set up a listener for specific event.

Parameters:
Name Type Description
eventName string

The name of event what will be listened.

The callback that handles the emitted event.

Emit the specific event.

Parameters:
